# Portfólio - Diego Carneiro de Jesus

## Descrição
Landing page funcional criada para apresentar meu portfólio pessoal, reunindo informações sobre minha formação, habilidades e projetos em desenvolvimento web.

## Seções da página
- **Sobre mim**: breve apresentação e foto de perfil.
- **Habilidades**: habilidades técnicas e comportamentais.
- **Projetos**: lista de projetos com nome, descrição, link de demonstração e tecnologias utilizadas.
- **Contato**: informações de contato e formulário.

## Projetos em destaque
### 1) Jogo da Velha (Tic-Tac-Toe)
- **Descrição**: Jogo feito com HTML, CSS e JavaScript, com detecção de vitória/empate, destaque das casas vencedoras e animação na mensagem de vitória.
- **Demo**: https://diegocjc26.github.io/Jogo-da-velha/
- **Tecnologias**: HTML, CSS, JavaScript

### 2) Portfólio (este site)
- **Demo**: https://diegocjc26.github.io/site/
- **Tecnologias**: HTML, CSS

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript (para interações/animações)

## Como executar o projeto
1. Baixe os arquivos do projeto.
2. Abra o arquivo `index.html` no navegador.
